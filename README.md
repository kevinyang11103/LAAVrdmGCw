# 项目简介 python1150

本项目是基于Python的高校学生学业预警系统，采用了Django框架和MySQL数据库，构建了一个前后端分离的Web网站。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**:
  - Python 3.x
  - MySQL 5.x
- **技术框架**: Python、Django、MySQL

## 功能模块

### 系统角色

系统设计包括两个角色：管理员与学生。

### 前台系统

- **首页**: 展示系统的主要信息和导航。
- **登录**: 学生和管理员可以登录系统。

### 后台管理系统

#### 管理员端

- **管理员登录首页**: 管理员登录后可查看系统主页面，包含预警分析、学生信息管理、学生成绩管理等功能模块。
- **预警分析管理**: 管理员可查看不同颜色标识的预警信息，点击学生姓名可查看详情，包括姓名、成绩、入学年份、学籍状态及预警类型。
- **学生管理**: 管理员可查看所有学生信息，以列表形式展示学生的学号、姓名、考试分数、学分、学期等，并可进行修改和删除操作。
- **学生成绩管理**: 管理员可查看学生的具体成绩信息，新增学生的成绩信息以及对存量的成绩信息进行修改。

#### 学生端

- **学生个人信息管理**: 学生可查看和维护个人信息，制定学习计划以改进预警情况。
